# Food Project

A responsive static landing page for a premium food delivery service called **Omifood**. The project is built as a visual demo website with sections for features, meals, workflow, locations, testimonials, pricing plans, and user signup.

## Features

- Responsive layout using CSS grid and custom styling
- Smooth anchor scrolling between page sections
- Animated scroll effects powered by AOS
- Interactive navigation for quick access to sections
- Meals showcase, city highlights, customer testimonials, and pricing plans
- Simple contact/signup form layout (static, no backend processing)

## Tech Stack

- HTML5
- CSS3
- jQuery
- AOS (Animate On Scroll)
- Ionicons for icons
- Google Fonts (Lato)

## Files and Structure

- `index.html` – main landing page
- `resources/css/style.css` – custom styles for the page
- `vendors/css/normalize.css` – browser reset stylesheet
- `vendors/css/grid.css` – grid system for layout
- `vendors/css/queries.css` – responsive media query rules
- `resources/img/` – image assets used on the page

## Usage

1. Open `index.html` in a web browser.
2. Use the navigation links to jump to sections such as Food Delivery, How It Works, Cities, and Sign-up.
3. Scroll through the page to view meal photos, plan options, and customer testimonials.

## Notes

- This is a static front-end demo project.
- The signup form is present for layout purposes and does not submit data to a server.
- The page includes external scripts from CDN for icons, animations, and smooth scrolling.
